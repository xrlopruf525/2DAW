Para conectarlo debemos instalar PyMySQL==1.1.0

y posteriormente irnos a el archivo __init__.py que se encuentra junto a settings.py y añadir 

import pymysql
pymysql.install_as_MySQLdb()


Luego iremos a docker 

docker run --name midb -e MYSQL_ROOT_PASSWORD=Stock1 -e MYSQL_DATABASE=Stockdb -e 
MYSQL_USER=usuario -e MYSQL_PASSWORD=usuario123 -p 3306:3306 -d mysql:8

Con este comando crearemos el contenedor con mysql. 


A continuación pondremos:
docker exec -it midb mysql -u root -p

y insertaremos la contraseña que pusimos antes de root en mi caso Stock1.


Ahora creamos la bbdd: 
CREATE DATABASE Stockbd CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

Luego crearemos el usuario 

CREATE USER 'usuario'@'%' IDENTIFIED BY 'Stock';
GRANT ALL PRIVILEGES ON Stockbd.* TO 'usuario'@'%';
FLUSH PRIVILEGES;

el usuario se pone de esta forma pero luego al escribirlo sera solo usuario.
y le damos permisos y privilegios.

Despues de esto haremos un migrate y ya deberia estar listo.
